NiKa Chat


Description

Is a web based video application that I created for my wife and I, due to us being in a long distance relationship.

It functions basically like an open chat-room, where you can enter and when both are in there, it will create a video call.
It has text chat with the ability to paste in images, synchronized Youtube/Facebook/TikTok player as well as plex media server embedded.
For us it's a neat way to do the things we do together online everyday.

This was super fun to make! and there has been several version up to this one built with vueJS and now Angular.
We use it every single day, and it works for us, there is no demo, as most everything is hardcoded into the app, and also I did not include screenshots with her in it.
